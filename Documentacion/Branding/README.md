# Manual de Identidad de Marca — Sentido Migrante

Este directorio contiene la documentación visual e interactiva de la identidad de marca de **Sentido Migrante**.

## Contenido

- [index.html](file:///d:/Projects%20%28Local%29/Sentido_Migrante/Branding/index.html): Manual interactivo de marca en formato web HTML.

## Características del Manual

1. **Paleta de Colores Interactiva**: Presenta de forma individual el color dominante (Verde Bosque), los secundarios (Menta, Suculenta, Olivo), el de acento/contraste (Púrpura) y los neutros/fondos (Gamas de crema y gris azulado). Permite copiar los códigos HEX con un solo clic.
2. **Tipografía**: Detalla las familias tipográficas, jerarquías y combinaciones oficiales:
   - **Satoshi**: Tipografía principal para títulos, logotipos y display.
   - **Inter**: Tipografía secundaria para cuerpos de texto, descripciones y elementos funcionales.
3. **Sistema de Logotipo**:
   - **Logotipo Principal**: Combinación horizontal oficial de isotipo + texto.
   - **Versión Alternativa**: Disposición vertical/apilada para espacios reducidos.
   - **Isotipo / Monograma**: El símbolo por sí solo.
4. **Reglas de Aplicación básicas**:
   - Versión oficial sobre fondo claro.
   - Versión oficial sobre fondo oscuro.
   - Versión monocromática (blanco y negro puro).
5. **Componentes en Contexto**: Pequeños previews interactivos de cómo se aplican los colores, botones, badges, navbar y cards con un conmutador para probar en entornos claros y oscuros.
6. **Exportación a PDF**: Se ha diseñado un estilo de impresión (`@media print`) optimizado en el propio archivo HTML para que al presionar el botón **"Guardar como PDF / Imprimir"** en la barra lateral, se exporte como un documento PDF impecable y compaginado.

## ¿Cómo abrir y ver el Manual?

Puedes abrir el archivo [index.html](file:///d:/Projects%20%28Local%29/Sentido_Migrante/Branding/index.html) directamente en cualquier navegador web moderno (Chrome, Safari, Firefox, Edge) para interactuar con él, o guardarlo directamente como PDF.
