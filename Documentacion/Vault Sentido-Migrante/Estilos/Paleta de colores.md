# Paleta de Colores — Sentido Migrante

Documento de referencia para la paleta de colores de la identidad visual de **Sentido Migrante**.

## 1. Colores de Acento (Actualizados)
Colores utilizados para llamados a la acción, elementos interactivos y contraste estratégico.

* **Rosa Viejo**
  * Hex: `#BB8084`
  * Variable: `--color-rosa-600`
  * Uso: Acento principal, botones interactivos, destacar categorías.
* **Rosa Profundo**
  * Hex: `#665051`
  * Variable: `--color-rosa-800`
  * Uso: Contraste de acento, texto sobre fondos muy claros de acento.

---

## 2. Colores Primarios (Dominantes)
Inspirados en la naturaleza, transmiten estabilidad, calma y arraigo.

* **Verde Bosque**
  * Hex: `#516750`
  * Variable: `--color-bosque`
  * Uso: Botones principales, encabezados, identidad de marca.
* **Verde Bosque Oscuro**
  * Hex: `#3F513E`
  * Variable: `--color-bosque-dark`
  * Uso: Hover de botones principales, textos destacados.

---

## 3. Colores Secundarios
Colores de apoyo para tarjetas, fondos suaves y etiquetas.

* **Verde Suculenta**
  * Hex: `#A2BC97`
  * Variable: `--color-suculenta`
* **Verde Olivo**
  * Hex: `#647453`
  * Variable: `--color-olivo`
* **Verde Menta**
  * Hex: `#DAEDDF`
  * Variable: `--color-menta`

---

## 4. Fondos y Neutros
Gamas de crema y gris para lectura cómoda y calidez visual.

* **Crema Claro**: `#FCFBF9` (`--color-cream-50`)
* **Crema Medio**: `#F9F7F3` (`--color-cream-100`)
* **Crema Oscuro / Arena**: `#F2EFE8` (`--color-cream-200`)
* **Gris Azulado Profundo**: `#1C1917` (`--color-bluegrey-900`)
